# Bezawit Tsige's Portfolio

Welcome to my personal portfolio website! This site showcases my background, interests, and projects as a passionate tech enthusiast and first-year undergraduate student at Addis Ababa University.

## Features

- **About Me:** Learn more about who I am and my journey in tech.
- **Education Background:** Details about my academic path and a link to view my CV.
- **Interests:** Discover what excites me in technology and beyond.
- **Projects:** Explore some of the projects I have worked on, with links to code or live demos.
- **Contact:** Find ways to get in touch with me.

## How to View

1. Clone or download this repository.
2. Make sure `index1.html` and `style.css` are in the same folder.
3. Open `index1.html` in your web browser to view the website.
